# 📊 Análisis de Ventas con Python

## 📌 Descripción del Proyecto

Este proyecto analiza datos de ventas de una empresa utilizando Python, pandas y matplotlib para identificar patrones de negocio, categorías más rentables y productos que generan pérdidas.

El objetivo principal es realizar un análisis exploratorio de datos (EDA) para obtener insights que ayuden a mejorar la rentabilidad y la toma de decisiones empresariales.

---

## 🔍 Objetivos del Análisis

Durante el proyecto se analizaron las siguientes preguntas de negocio:

* ¿La empresa es rentable?
* ¿Qué categorías generan más ventas y ganancias?
* ¿Qué productos son los más rentables?
* ¿Qué productos generan pérdidas?
* ¿Cómo afectan los descuentos a la rentabilidad?
* ¿Existen categorías con bajo margen de ganancia?

---

## 📈 Insights Principales

* La categoría Tecnología presenta las mayores ventas y ganancias.
* Furniture presenta altas ventas pero bajo margen de ganancia.
* Office Supplies genera mejores márgenes que Furniture.
* Los descuentos altos generan pérdidas significativas.
* Existe una fuerte relación negativa entre descuentos y profit.
* Algunos productos presentan pérdidas importantes y deberían ser revisados.
